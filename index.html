Cross Hatch Generator

Description:

This was inspired by styles of sketching, AKA cross hatching, found in the story boards of comic books, films, etc. With this generative javascript application, the user has the power to manipulate the elements of the grid to make patterns of their own. Each toggle to the right has the ability to change the sequence of events that occurs.

Parameters:

  • ScALE: This controls the Overall width of each individual line
  • Distance x: This controls the distance between each line via X-axis
  • Distance Y: This controls the distance between each line via Y-axis
    bColor: this changes the background color of the canvas
    Color: This changes the color of the lines
    Xspeed: This controls the speed of line movement via x-axis
    yspeed: This controls the speed of line movement via y-axis
    zspeed: This controls the speed of line movement via z-axis
